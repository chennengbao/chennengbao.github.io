## API文档 

### 数组去重（一） 

    $cnb.uniqueArrByFor(arr)

处理数组去重。

#### 参数 

##### arr 

类型：数组<br>
默认：无默认值<br>
描述：需要去重的数组　　

##### return 

返回去重后的数组

#### 使用方法示例 

	var arr = [1, 10, 2, 4, 3, 6, 10, 3, 4, 99];
	$cnb.uniqueArrByFor(arr);

	结果：[1, 10, 2, 4, 3, 6, 99]
