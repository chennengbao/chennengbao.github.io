API文档

1、uniqueArrByFor 数组去重

$cnb.uniqueArrByFor(arr)

处理数组去重。

参数

arr

类型：数组
默认：无默认值
描述：需要去重的数组　　

return

返回去重后的数组

使用方法示例

var arr = [1, 10, 2, 4, 3, 6, 10, 3, 4, 99];
$cnb.uniqueArrByFor(arr);

结果：[1, 10, 2, 4, 3, 6, 99]

支持程度

IE6+、iOS、Android

Bug & Tip

无

5、rectangleContainPoint 判断一个点是否在矩形内

$cnb.rectangleContainPoint(arr, judgePoint)

判断一个点是否在矩形内。

参数

arr

类型：数组
默认：无默认值
描述：矩形的四个点，顺时针方向
示例：

arr[
    {x: 3, y: 3},
    {x: 3, y: 6},
    {x: 6, y: 6},
    {x: 6, y: 3}
];

judgePoint

类型：对象
默认：无默认值
描述：需要判断的点
示例：

{
    x: 5, 
    y: 3
}

return

返回结果（true | false）

使用方法示例

arr.push({x: 3, y: 3});
arr.push({x: 3, y: 6});
arr.push({x: 6, y: 6});
arr.push({x: 6, y: 3});
$cnb.rectangleContainPoint(arr, {x: 5, y: 3});

结果：true

支持程度

IE6+、iOS、Android

Bug & Tip

无

8、commentHandleToString 将注释处理成字符串

$cnb.commentHandleToString(commentFn)

将注释处理成字符串。

参数

commentFn

类型：函数
默认：无默认值
描述：含有注释的回调函数　

return

返回注释的内容

使用方法示例

 var str = $cnb.commentHandleToString(function() {
    /* <dl class="{{author}}">
           <dt>{{cnb}}</dt>
           <dd>{{h5course}}</dd>
       </dl> *\/
});

结果：
<dl class="{{author}}">
   <dt>{{cnb}}</dt>
   <dd>{{h5course}}</dd>
</dl>

支持程度

IE6+、iOS、Android

Bug & Tip

目前只支持块注释

9、resolveString 解析含有特殊标记的字符串

$cnb.resolveString(strFn, dataObj)

解析含有特殊标记的字符串。

参数

strFn

类型：函数
默认：无默认值
描述：含有注释的回调函数　

dataObj

类型：对象
默认：无默认值
描述：替换特殊标记的数据　

return

返回解析后的结果

使用方法示例

var str = $cnb.resolveString(function() {
    /* <dl class="{{author}}">
           <dt>{{cnb}}</dt>
           <dd>{{h5course}}</dd>
       </dl> *\/
    }, {
       author: 'wrap',
       cnb: '陈能堡',
       h5course: 'HTML5'
});

结果：
<dl class="wrap">
   <dt>陈能堡</dt>
   <dd>HTML5</dd>
</dl>

支持程度

IE6+、iOS、Android

Bug & Tip

目前只支持块注释
